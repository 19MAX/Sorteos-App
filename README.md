# Proyecto Sorteos

Aplicación web desarrollada en CodeIgniter que permite gestionar sorteos de productos mediante boletos numerados.

## Características

* Creación de sorteos por producto
* Configuración de número de boletos por sorteo
* Compra o reserva de boletos para participantes

## Tecnologías

* PHP
* CodeIgniter
* MySQL
* JavaScript

## Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/19MAX/Sorteos-App
```

2. Entrar al proyecto:

```bash
cd sistema-sorteos
```

3. Configurar variables de entorno:

* Archivo `.env`
* Conexión a base de datos

4. Ejecutar las migraciones

```bash
php spark migrate --all
```

5. Ejecutar servidor:

```bash
php spark serve
```

## Uso básico

1. Crear un producto
2. Definir cantidad de boletos
3. Publicar el sorteo
4. Ejecutar sorteo
